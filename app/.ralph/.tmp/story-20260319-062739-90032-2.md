### US-002: Heart icon on asset cards
Status: in_progress
Depends on: US-001

Description:
As a content author, I want to favourite an asset with one click from the grid so I can build a shortlist without opening the detail modal.

Acceptance Criteria:
- [ ] AssetCard receives isFavourited (boolean) and onFavouriteToggle (function) props
- [ ] A heart button is rendered top-left of the thumbnail image, position absolute, top 8px left 8px
- [ ] Button: 28×28px, border-radius 50%, background rgba(0,0,0,0.35), border none, cursor pointer, display flex, align-items center, justify-content center
- [ ] Icon: ♡ (not favourited) or ♥ (favourited), font-size 14px
- [ ] Not favourited: icon color #FFFFFF
- [ ] Favourited: icon color #EF4444
- [ ] On click: calls e.stopPropagation() then onFavouriteToggle(asset.id, { display_title: asset.display_title, thumbnail_path: asset.thumbnail_path, cdn_url: asset.cdn_url })
- [ ] On toggle: button animates transform scale(1.2) → scale(1) over 150ms
- [ ] BrowsePage passes isFavourited={isFavourited(asset.id)} and onFavouriteToggle={onFavouriteToggle} to each AssetCard
- [ ] Example: clicking ♡ on a card immediately shows ♥ in red
- [ ] Negative case: clicking ♥ on an already-favourited card removes it and shows ♡
- [ ] npm run lint passes
- [ ] npm run build passes
