### US-002: Serve DAM media files via Express static route
Status: in_progress
Depends on: US-001

Description:
As a content author, I want to see asset thumbnails in the browse grid so I can visually identify assets.

Acceptance Criteria:
- [ ] In server/src/index.js, add a static file route: app.use('/api/assets/media', express.static(path.resolve(__dirname, '..', '..', process.env.DATA_DIR || '../Data/royal')))
- [ ] Import path from 'path' at the top of index.js if not already imported
- [ ] Route is registered before any error-handling or 404 middleware
- [ ] GET http://localhost:3001/api/assets/media/promotions/asia/arabian-gulf/dubai-arabian-gulf-emirates-burj-al-arab-skyline.jpg/_jcr_content/renditions/cq5dam.thumbnail.319.319.png returns 200 with image content-type
- [ ] Supertest test: GET /api/assets/media/ships/allure/allure-of-the-seas-aerial-sailing-blue-sky.JPG/_jcr_content/renditions/cq5dam.thumbnail.319.319.png returns 200 or 404 (not 500) — verifies route is wired up
- [ ] Negative case: GET /api/assets/media/nonexistent/path.png returns 404, not 500
