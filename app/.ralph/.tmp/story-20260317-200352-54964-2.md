### US-002: Independent scroll containers for sidebar and grid
Status: in_progress
Depends on: US-001

Description:
As a user, I want the filter sidebar and asset grid to scroll independently so filters stay accessible while I browse.

Acceptance Criteria:
- [ ] Sidebar div has overflow-y:auto, height:100%, fixed width 240px
- [ ] Main content div has overflow-y:auto, height:100%, flex:1
- [ ] Scrolling the grid does not affect the sidebar scroll position
- [ ] Scrolling the sidebar does not affect the grid scroll position
- [ ] Results count text ('Showing X of Y assets') is inside the main scroll container, above the grid
- [ ] No scrollbar appears on the body/html element
- [ ] Vitest test: layout renders without body-level overflow
