### US-004: AssetDetailModal — variant thumbnail strip
Status: in_progress
Depends on: US-002

Description:
As a content author, I want to see and switch between all variants of an asset in the detail modal so I can find the right format without closing the modal.

Acceptance Criteria:
- [ ] On open, if asset has variant_group_id set, call getAssetVariants(id) and store variants in local state
- [ ] If variants.length > 1, render a variant strip in the left panel below the main image
- [ ] Strip header: 'Variants (N)' label, 12px grey text
- [ ] Strip: horizontal scrollable row (overflow-x:auto, display:flex, gap:8px)
- [ ] Each thumb: 72px × 72px img (object-fit:cover, border-radius:4px, cursor:pointer), with a channel label below (10px, #6b7280)
- [ ] Active/selected thumb has 2px solid navy border
- [ ] Clicking a thumb calls getAsset(variantId), updates asset state — all modal content updates (image, title, metadata, CDN URL, quality issues)
- [ ] Initially selected thumb is the currently opened asset
- [ ] If getAssetVariants returns [] or errors, strip is not rendered
- [ ] Vitest test: modal fetches variants on open when asset has variant_group_id
- [ ] Vitest test: clicking a variant thumb calls getAsset with the variant's id
- [ ] Vitest test: no strip rendered when variants array is empty
