### US-004: Compact modal — collapsible metadata accordion, always-visible tags, clean key-value layout
Status: in_progress
Depends on: US-003

Description:
As a content author, I want the detail modal to open in a compact state with a clean metadata layout so I can see key information immediately without column headers or section labels cluttering the view.

Acceptance Criteria:
- [ ] Remove the standalone 'Metadata' section heading text entirely — the accordion header replaces it
- [ ] Remove the table column headers ('Field', 'Value', 'Original', 'Enriched') — no thead, no header row
- [ ] Replace the HTML table with a clean div-based key-value list: each row is a flex row with label on left and value on right
- [ ] Label: 11px, font-weight 600, text-transform uppercase, letter-spacing 0.04em, color #9CA3AF, width 110px, flex-shrink 0
- [ ] Value: 13px, font-weight 400, color #111827 (or #9CA3AF for empty/not enriched), flex 1
- [ ] Rows separated by padding only (8px top/bottom per row) — no border lines between rows
- [ ] The metadata section (toggle + key-value list) is wrapped in a collapsible accordion
- [ ] Accordion header: full-width row, label 'Details' (13px/600, color #374151) on left, chevron '▼'/'▲' (12px, color #9CA3AF) on right, cursor pointer, padding '8px 0', border-top '1px solid #F3F4F6'
- [ ] Default state: collapsed (key-value list and toggle not rendered)
- [ ] Clicking the header toggles open/closed
- [ ] A Tags summary row is always visible above the accordion header (never inside the accordion)
- [ ] Tags summary: small 'Tags' label (11px/600 uppercase muted), then display_tags parsed and rendered as navy pill chips (bg rgba(0,32,91,0.07), color #00205B, border-radius 100px, font 11px/500, padding 2px 8px, gap 4px, flex-wrap wrap)
- [ ] If display_tags is empty or null, the tags summary row is not rendered
- [ ] Right panel order from top: title+actions row, tags summary (if present), 'Details' accordion header, [expanded: Enriched/Original toggle + key-value list], CDN URL section, Enrich button
- [ ] Accordion open/closed state resets to closed each time the modal opens
- [ ] Example: opening modal shows title, action buttons, tag pills, a 'Details ▼' row — clean and uncluttered
- [ ] Example: clicking 'Details ▼' expands to show the toggle and clean key-value rows with no column headers
- [ ] Negative case: asset with no tags → tags row not rendered; accordion still works correctly
- [ ] npm run lint passes
- [ ] npm run build passes
