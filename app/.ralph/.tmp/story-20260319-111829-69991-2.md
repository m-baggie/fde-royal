### US-002: Smart search UI — mode toggle, prompt input, explanation banner
Status: in_progress
Depends on: US-001

Description:
As a content author, I want to switch to smart search mode and type a natural language prompt so I can find assets by describing what I need.

Acceptance Criteria:
- [ ] SearchBar.jsx (or BrowsePage.jsx) renders a '✦ Smart' toggle pill to the right of the search input
- [ ] Toggle inactive style: border '1px solid #E5E7EB', background transparent, color #9CA3AF, border-radius 100px, font 11px/600, padding 3px 10px, cursor pointer
- [ ] Toggle active style: background #001B6B, color #FFFFFF, border-color #001B6B
- [ ] When smart mode is active: input placeholder changes to 'Describe what you're looking for...' and input border becomes 1.5px solid #001B6B
- [ ] When smart mode is active: typing does NOT trigger debounced search — only Enter key or a small '→' submit button (appears inside the input on the right when there is text) triggers the search
- [ ] Clicking '→' or pressing Enter calls smartSearch(query) from the API module
- [ ] While waiting for response: show '✦ Thinking...' in the explanation banner area (above results count), results are not cleared during loading
- [ ] On response: update assets grid with returned assets, show explanation banner above results count
- [ ] Explanation banner: background rgba(0,27,107,0.06), border-radius 8px, padding 8px 12px, font 12px color #001B6B. Text: '✦ ' + explanation
- [ ] Clearing the search input hides the explanation banner and resets to normal search
- [ ] Toggling off smart mode clears the explanation banner and triggers a normal search with the current input value
- [ ] If explanation is null (fallback): no banner shown, results display normally
- [ ] BrowsePage passes smartMode state and explanation to the search area components
- [ ] Example: typing 'happy families by the pool' and pressing Enter shows assets + banner '✦ Showing families enjoying pool activities on deck'
- [ ] Negative case: if smart search returns 0 results, banner still shows with explanation and results count shows 0
- [ ] npm run lint passes
- [ ] npm run build passes
