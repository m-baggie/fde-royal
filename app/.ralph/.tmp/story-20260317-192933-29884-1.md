### US-001: Fixed header + search bar layout shell
Status: in_progress
Depends on: None

Description:
As a user, I want the header and search bar to always be visible so I can search without scrolling back to the top.

Acceptance Criteria:
- [ ] App.jsx renders a flex-column wrapper that fills 100vh with overflow:hidden
- [ ] Header row is 48px tall, navy background, contains logo (left) and Upload button (right)
- [ ] Search bar row is below header, white background with bottom border, contains SearchBar component at full width
- [ ] Body row below search takes flex:1, overflow:hidden, and contains sidebar + main in a flex row
- [ ] body/html overflow:hidden is set on mount and cleaned up on unmount
- [ ] Vitest test: BrowsePage renders a header element and a search input
- [ ] Vitest test: sidebar and main content area both exist in the rendered output
