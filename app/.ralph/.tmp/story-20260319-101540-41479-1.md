### US-001: AssetDetailPanel component — vertical layout with image at top
Status: in_progress
Depends on: None

Description:
As a content author, I want to see asset details in a side panel with the image at the top so I can evaluate the image and read metadata without losing context of the grid.

Acceptance Criteria:
- [ ] client/src/components/AssetDetailPanel.jsx is created
- [ ] Panel container: width 400px, height 100%, border-left '1px solid #E5E7EB', background #FFFFFF, overflow-y auto, flex-shrink 0, position relative
- [ ] Close button: position absolute, top 12px, right 12px, 32×32px, background none, border none, fontSize 20px, color #9CA3AF, cursor pointer, z-index 1
- [ ] Image zone: width 100%, aspect-ratio 16/9, object-fit cover. Uses web_image_path or thumbnail_path via /api/assets/media/ route. Shows grey placeholder if no image.
- [ ] Variant strip: rendered below the image when variants.length > 1 — horizontal scrollable strip of 64×64px thumbnails, same style as current modal
- [ ] Content zone: padding 20px, display flex, flex-direction column, gap 16px
- [ ] Title row: display flex, align-items flex-start, gap 8px. Title: 16px/700 color #111827, flex 1. Action buttons: '↓ Download' (text+icon, navy), favourite heart (♥/♡, toggles red when active), share (↗), and delete (🗑 — admin only). The favourite and delete buttons are icon-only 32×32px; Download and Share have text labels.
- [ ] Tags row: always visible. 'Tags' label (11px/600 uppercase muted) above navy pill chips. Hidden if no tags.
- [ ] Details accordion: same collapsible accordion as specified in prd-modal-ux US-004 — label 'Details', collapsed by default, contains Enriched/Original pill toggle and key-value metadata rows
- [ ] CDN URL section: shown when asset.cdn_url exists — code block + copy button
- [ ] Enrich button: shown when enrichment_source is null or pending
- [ ] All data fetching logic copied from AssetDetailModal: getAsset on selectedAssetId change, getAssetVariants if variant_group_id set, enrichAsset, handleCopy, handleDelete, handleVariantClick
- [ ] Escape key listener closes panel via onClose()
- [ ] Example: clicking an asset card shows the panel with image at top and title below
- [ ] Negative case: asset with no image shows grey placeholder in the image zone
- [ ] npm run lint passes
- [ ] npm run build passes
