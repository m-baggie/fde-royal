### US-003: enrich-all.js reseed mode — wipe and re-enrich all AI-enriched assets
Status: in_progress
Depends on: US-002

Description:
As a developer, I want to run the bulk enrichment script to wipe all existing AI enrichment and re-run with the new prompt so the DB is clean and consistent.

Acceptance Criteria:
- [ ] enrich-all.js accepts a --reseed flag: node enrich-all.js --reseed
- [ ] When --reseed is passed: first runs UPDATE to NULL all enriched_* fields and set enrichment_source=NULL for all assets where enrichment_source IS NOT NULL AND enrichment_source != 'user-edited'
- [ ] After wipe, proceeds to enrich all assets where enrichment_source IS NULL (same as normal run)
- [ ] Prints 'Reseed mode: wiped X assets' before starting enrichment
- [ ] Normal run (no --reseed) skips wipe — only enriches assets where enrichment_source IS NULL
- [ ] Script prints final summary: Enriched: X | Failed: Y | Skipped: Z
